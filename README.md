# Ebike
un progetto che serve per trovare, visualizzare e selezionare Ebike in cpp.
il progetto è un compito della scuola di cui si esercito sull'uso dello struct.
funzione:
1.con xy dell'utente e quel di bici calcola la distanza tra entrambi
2.ordina automaticamente la lista di Ebike da più vicina a più lontana ed esclude non disponibili
3.calcola del tempo e soldi che utente deve pagare dopo aver finito a usare.
